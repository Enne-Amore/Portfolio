Como o meu foco em procurar emprego é trabalhar como ume desenvolvedore front-end, então costumo construir projetos para mostrar o meu potencial e conhecimento a quem pode desejar me contratar algum dia. Dessa forma, faço websites focados em mostrar a parte visual e interativa do site.

A plataforma que mais uso para codar se chama Visual Studio Code a qual é um ótimo ambiente de desenvolvimento integrado de ser utilizado em minha área. Ademais, utilizo GitHub com o recurso GitHub Pages ativado para hospedar os meus websites a fim de todes que têm acesso à internet poderem ver.
